# A1 - Engine Repair 

# Exam Tips
- Symptom → Component → Tool → Procedure/Spec
- Verify concerns/road test, research TSBs/recalls/ADAs
- No-start: fuel/ignition/spark/compression sequence
- Hybrid/HEV: high-voltage disconnect procedures
- Vacuum tests, relative compression, cylinder contribution

### Cylinder Head and Valve Train
- Cylinder heads (cracks, warpage, passages)
- Valves, seats, springs, retainers, locks, guides
- Valve stems, seals, pushrods, rocker arms
- Lifters, camshafts (lobes, journals, timing), VVT/VVL components

### Engine Block
- Engine block (cracks, warpage, bores)
- Pistons, rings, pins
- Connecting rods, bearings
- Crankshaft (journals, endplay, thrust), main bearings
- Vibration dampers, auxiliary shafts, cylinder deactivation
- Flywheels/ring gears, dual-mass flywheels (inspection/runout)

### Lubrication System
- Oil pumps (gears, rotors, relief valves, pickup)
- Oil filters, coolers, pan, passages

### Cooling System
- Radiator, heater core, cap, recovery system
- Hoses, thermostat, water pump
- Fan, clutch, shroud, belts, tensioners

### Fuel, Ignition, Electrical, Exhaust
- Fuel injectors, intake manifold, air filters, turbo/superchargers
- Cranking system, PCV valve/hoses
- Ignition components (coils, plugs, wires)
- Exhaust manifolds

### Additional Components
- Timing belts/chains (inspection, replacement, tensioning)
- EGR systems (valves, passages, controls)
- Sensors (cam/crank position, knock, MAP/MAF, coolant temp)
- Engine mounts and flexplates
- Core plugs, gallery plugs, O-rings/seals (general)
- Drive pulleys/idlers (beyond belts)
- Gaskets/seals during reassembly
- Oil galleries/passages (block/head cleaning)
- Balance shafts (inspection)
- Cylinder deactivation solenoids/lifters
- Piston oil jets, variable stroke engines
- Head gasket multi-layer steel (MLS) types

### Diagnostic Tools
- Compression tester, leak-down tester, vacuum gauge
- Scan tools (OBD-II for DTCs, live data), oscilloscope
- Cylinder power balance tester
- Oil pressure gauges
- Stethoscopes (for noises)
- Thermometer/infrared temp gun (hot spots)
- Fuel pressure gauge

### Measurement Tools
- Micrometers (inside/outside), dial bore gauge, telescoping gauges
- Feeler gauges (valve clearance), dial indicators (cam lift, runout)
- Torque wrench (inch/pounds, foot/pounds), plastigage

### Service Tools
- Timing tools (chain/belt tensioners, alignment kits)
- Valve spring compressor, piston ring expander/compressor
- Cylinder hone, head resurfacing equipment, pullers (harmonic balancer)

### Performance Symptoms
- Hard starting or no-start
- Rough idle or stalling
- Hesitation, lack of power, or poor acceleration
- Engine misfire (single/multiple cylinders)
- Excessive vibration or rough running

### Noise Symptoms
- Knocking or pinging (detonation)
- Ticking or tapping (valvetrain)
- Rattling (timing chain, lifters)
- Whining (pumps, bearings)
- Backfiring or popping

### Leak/Consumption Symptoms
- External oil leaks
- Coolant leaks or overheating
- Excessive oil/fuel/coolant consumption
- White/blue/black exhaust smoke
- Milky oil (head gasket failure)
- Hydro-lock (liquid in cylinders)

### Warning Symptoms
- Check engine light (CEL/DTCs)
- Low oil pressure light
- High temperature gauge
- Burning smells (oil, coolant, fuel)

### Diagnosis Flow
- Road test → visual inspection → DTC scan → compression/leakdown test → power balance
- Noise: stethoscope pinpoint → component removal for inspection

### Measurement Specs
- Cylinder bore taper/out-of-round (<0.005" typical)
- Valve clearance (intake 0.008-0.012", exhaust 0.010-0.014" hydraulic)
- Bearing clearance (0.001-0.003" mains/rods via plastigage)

### Service Steps
- Head: pressure test → resurface if warp >0.003"/6" → lap valves
- Block: hone cylinders (45° crosshatch) → ring gap 0.016-0.077"
- Timing: align marks → tensioner lock → rotate 2 full turns verify

# A6 - Electrical/Electronic Systems

### Examp Tips
- Voltage drop <0.5V everywhere (golden rule)
- Parasitic draw: <50mA after 30min sleep
- Charging: 13.8-14.4V running engine
- Flashcards: Symptom → Tool → Spec (e.g., "No crank → Voltage drop → <0.5V solenoid")

### Battery System
- Batteries (12V lead-acid, AGM, flooded, spirals)
- Battery cables, terminals, hold-downs
- State-of-charge testers, load testers

### Starting System
- Starter motors (PM, series-wound), solenoids
- Starter drives (one-way clutches, overrunning)
- Starter relays, ignition switch, neutral safety switch

### Charging System
- Alternators (diode trios, regulators, rotors, stators)
- Voltage regulators (internal/external), drive belts
- Rectifiers, brushes, slip rings

### Lighting Systems
- Headlights (halogen, HID, LED), bulbs, sockets
- Taillights, brake lights, turn signals, hazards
- Headlight adjusters, multifunction switches

### Gauges/Instruments
- Instrument cluster (stepper motors, gauges)
- Speedometer sensors (VSS), tachometers
- Warning lights (ABS, airbag, CEL), driver info displays

### Body Electrical
- Power windows, door locks, mirrors (motors, switches)
- Wiper motors (interval, park), washer pumps
- Horns, power accessories, CAN bus networks
- LIN bus (single-wire networks), gateway modules
- LED driver modules, adaptive headlights
- Battery management systems (BMS) in hybrids

### General Circuits
- Fuses, fusible links, circuit breakers, relays
- Wiring harnesses, connectors, grounds
- Multiplex systems, BCM (body control modules)

### Circuit Testing Tools
- Digital multimeter (DMM) (voltage, resistance, current, diode test)
- Test light (self-powered, powered)
- Voltage drop tester (across cables, connections)

### Battery/Charging Tools
- Battery load tester (carbon pile, conductance)
- Carbon pile tester (alternator output)
- Amp clamp (parasitic draw, starter current)

### Starting/Lighting Tools
- Starter draw tester (amp clamp)
- Bulb socket tester, headlight aimer
- Relay puller/tester, circuit breaker tester

### Advanced/Network Tools
- Oscilloscope (waveforms for sensors, alternator AC ripple)
- Scan tool (module communication, CAN bus)
- Wiring diagram software/piercing probes
- Logic probe (high/low/undefined signals)

### Battery/Starting Symptoms
- Slow cranking or no crank
- Clicking sound (solenoid engages, no spin)
- Dead battery overnight (parasitic draw)
- Dim headlights during cranking

### Charging Symptoms
- Battery light on dash
- Overcharging (>15V) or undercharging (<13.5V)
- Flickering lights or erratic gauges
- Burning alternator belt smell

### Lighting Symptoms
- One headlight out (bulb/ground)
- Hyper-flashing turn signals
- Brake lights work, hazards don't (flasher relay)
- Interior lights dim/fading

### Gauges/Instruments Symptoms
- Erratic speedometer/tachometer
- Warning lights stay on (ABS, airbag, CEL)
- Fuel/oil/temp gauges pegged zero/max
- Driver info center blank

### Body Electrical Symptoms
- Power windows slow/one side dead
- Intermittent wipers park wrong
- Horn weak/no sound
- Door locks click but don't operate

### General Circuit Symptoms
- Intermittent operation (heat/cold related)
- Multiple unrelated circuits fail (ground issue)
- CAN bus codes (U-codes)

### Key Diagnostic Procedures
- Voltage drop test (<0.5V cables/connections, <0.2V grounds)
- Parasitic draw test (<50mA key off after 30 min sleep)
- Charging system test (13.8-14.4V running, ripple <0.5V AC)

### Battery/Starting Tests
- Battery load test (9.6V+ at half CCA rating)
- Starter current draw (150-250A typical)
- Solenoid voltage drop (<0.5V engage circuit)

### Circuit/System Verification
- Continuity test (0 ohms good wire, OL open)
- Relay test (click + 12V output when energized)
- CAN bus check (2.5V high/low signal with scope)

### Service Specs
- Alternator diode ripple (<0.5V AC)
- Bulb resistance match (check specs)
- Multiplex resistance (<5 ohms network)

# A7 - Heating & Air Conditioning

### Examp Tips

### Refrigeration System Components
- Compressor (clutch, swash plate, scroll, variable displacement, solenoid control, relief valves)
- Condenser (parallel flow, tube-and-fin, subcooling)
- Evaporator (core, expansion valve/TXV/orifice tube integration)
- Receiver-drier/accumulator (desiccant bag, sight glass)
- Hoses, lines, fittings, O-rings, service valves/ports
- Expansion devices (TXV, fixed orifice, electronic)

### Heating System Components
- Heater core, matrix, housing/doors
- Engine coolant hoses/pipes, heater control valve
- Blower motor/resistor/relay, fan blades
- Temperature blend/mix doors (cable, vacuum, servo)
- Radiator, thermostat, water pump (engine/heater circuits)

### Operating Controls
- Manual controls (sliders, knobs, vacuum actuators)
- Automatic climate control (sensors, actuators, ECUs)
- Pressure switches (high/low/trinary), cycling switch
- Temperature sensors (ambient, in-car, evaporator, discharge air)
- Thermostat (fixed/variable electronic)

### Electrical/Electronic Components
- Clutch relay, blower relay/resistors, fuses
- Wiring harnesses, connectors, protection devices
- Actuators/servos (door position), stepper motors
- ECM/AC control module inputs/outputs
- Cabin air filters, fresh/recirc doors

### Diagnostic/Service Components
- Manifold gauge set, vacuum pump, recovery/recycling unit
- Leak detectors (electronic, UV dye, nitrogen)
- Flush kits, oil injectors (PAG/ester matching)

### Performance Diagnostic Tools
- Manifold gauge set (high/low side pressures, superheat/subcooling)
- Thermometer (vent temp, discharge air, evaporator/ambient)
- Tachometer (engine RPM, compressor clutch speed)
- Anemometer (airflow velocity at vents)

### Refrigerant Service Tools
- Recovery/recycling/recharge machine (R-134a/R-1234yf compatible)
- Vacuum pump (deep vacuum to 500 microns)
- Refrigerant identifier (purity analyzer)
- Leak detectors: electronic sniffer, UV dye kit, fluorescent developer, nitrogen pressure test

### Electrical/Electronic Tools
- Digital multimeter (DMM) (voltage, resistance, current draw on clutch/blower)
- Scan tool (ATC DTCs, actuator tests, sensor data)
- Test light/continuity tester (relays, switches)
- Oscilloscope (clutch waveform, sensor signals)

### Mechanical Service Tools
- Flaring tool, swaging tool (lines/hoses)
- Compressor clutch puller/installer, oil injector
- Flush gun/kit (system flushing solvents)
- Spring compressor (TXV power element)

### A/C Performance Symptoms
- Insufficient cooling (warm vents, poor A/C at idle/highway)
- Low/high side pressures abnormal (both low: undercharge; both high: overcharge; low high/high low: restriction)
- Evaporator icing/freezing (low side line frost)
- Compressor not engaging/cycling erratically

### Heating Symptoms
- No heat/insufficient heat (cold air from heater)
- Overheating cabin/engine (stuck blend door, air in heater core)
- Inconsistent temps (cold spots, thermostat issues)

### Airflow Symptoms
- Low airflow from vents (clogged cabin filter, blower failure)
- Wrong air direction (stuck doors/actuators, vacuum leaks)
- Noisy blower (resistor, bearings, debris)

### Noise/Odor Symptoms
- Compressor noise (internal failure, clutch slip)
- Unusual smells (sweet/sour: evaporator mold; burning: electrical)
- Hissing/whistling (refrigerant leaks, expansion valve)

### Electrical/Warning Symptoms
- Clutch not engaging (low pressure switch, relay/fuse)
- ATC errors/DTCs (sensor faults, control head)
- Fan speeds limited (resistor, wiring)

### Diagnostic Procedures
- A/C performance test: manifold gauges (low 25-45 psi, high 2-2.5x ambient), vent temp (40-50°F), 5-10 min runtime, recirculation max AC
- Visual/road test: check vents, clutch cycling, airflow, TSBs/recalls
- Electrical checks: DMM on clutch voltage (12V), relay click, sensor resistance

### Refrigerant Service Procedures
- Recovery: use certified machine, full evacuation before opening
- Evacuate: vacuum pump to 500 microns, hold 30 min (no leaks)
- Leak test: electronic sniffer, UV dye, 150 psi nitrogen
- Recharge: weigh in exact spec (R-134a/R-1234yf), oil matching

### Component Service Procedures
- Compressor: clutch puller removal, oil drain/measure/add
- Expansion device: orifice tube replace (inline filter), TXV powerhead test
- Flush system: condenser/evaporator (avoid compressor), PAG/ester oils

### Heating Procedures
- Bleed air from heater core (heat on max, engine run)
- Blend door test: temp sweep, actuator voltage/position scan

## Safety & Regulations
- EPA 609: R-134a/R-1234yf handling, recovery certification
- High-voltage hybrids: disconnect orange cables, wait 10 min
- PPE: safety glasses, gloves for refrigerant/oil

## Diagnosis Flow
- Verify complaint → performance test (gauges/vents) → electrical checks → leak test → recover/evacuate → repair → recharge → verify

## Key Specs
- Superheat: 5-15°F at evaporator outlet
- Subcooling: 5-15°F at condenser outlet
- Ambient control: high side ~2.2-2.5x °F (ex: 200 psi @90°F)

# A8 - Engine Performance

### Study Tips
- Symptom → PID/data → Tool → Procedure
- "P0171 → STFT +15% → Fuel pressure gauge → 45 psi running"
- "P0302 → Scope cyl #2 → Injector pulse tester → Multimeter resistance"
- "P0420 → Backpressure gauge → Temp gun → >100°F inlet/outlet diff"
- "P0131 → O2 slow switching → Lab scope → 8-10Hz cross-counts"  ← ADD
- "High idle → Vacuum leak → Smoke machine → IAC/EGR passages"   ← ADD

### Ignition System
- Coils (coil-on-plug, distributor), ignition wires, spark plugs
- Crank/cam sensors, knock sensors, ignition module/control
- Distributor (cap, rotor, advance mechanisms)

### Fuel System
- Fuel pumps (in-tank, mechanical), injectors, rails, pressure regulators
- Fuel filters, throttle body (TBI), pressure test ports
- Vapor canister/purge valve, fuel tank/pressure sensor

### Air Induction/Exhaust
- Air filters, throttle body, intake manifold (gaskets, plenum)
- MAF/MAP sensors, IAC valve, turbo/supercharger
- Catalytic converters, mufflers, resonators, O2 sensors

### Emissions Controls
- EGR valve/system (vacuum, electric, cooler), PCV valve/hoses
- EVAP system (canister, vent valve, leak detection pump)
- Early fuel evaporation (EFE) valves

### Sensors/Electrical
- ECT, IAT, TPS, RPM sensors
- ECM/PCM, wiring harnesses, relays, fuses
- Data link connector (DLC), battery/alternator impact
- VVT solenoids/actuators, cam phasers
- Wideband O2 sensors, upstream/downstream locations
- GDI high-pressure fuel pumps (2000+ psi), carbon buildup

### Diagnostic Tool
- Scan tools (OBD-II bidirectional, live data, graphing)
- Multimeter (voltage, resistance, current for sensors/coils)
- Oscilloscope/lab scope (waveforms for crank/cam, O2, injectors)

### Fuel/Ignition Tools
- Fuel pressure gauge (low/high side, regulator test)
- Injector pulse tester/nozzle tester
- Spark tester (inline, adjustable gap), coil tester

### Emissions/Air Tools
- Smoke machine (EVAP leak detection)
- Exhaust backpressure gauge, vacuum gauge
- Lab scope for O2 sensor cross-counts

### Service Tools
- Nozzle cleaner (injectors), compression tester (wet/dry)
- Battery load tester (ECM power issues)
- Infrared temp gun (catalyst efficiency)

### Performance Symptoms
- Hard/no start, rough idle, stalling
- Hesitation, surging, lack of power
- Misfire (intermittent/random), poor acceleration
- Excessive vibration, poor fuel economy

### Emissions Symptoms
- Check engine light (CEL), DTCs (P0171/P0174 lean, P0300 misfire)
- Failed emissions (high HC/CO, low NOX)
- Black/white smoke, sulfur smell

### Fuel/Air Symptoms
- Rich/lean conditions (fuel trims >±10%)
- Vacuum leaks (hissing, high idle)
- Restricted exhaust (backpressure >2 psi)

### Electrical/Sensor Symptoms
- Erratic tachometer, dead miss at idle
- Poor cold/hot performance
- Catalyst efficiency codes (P0420)

### Diagnosis Flow
- Scan DTCs/freeze frame → verify drive cycle → monitor fuel trims (±10% threshold) → scope sensor waveforms → component test
- Emissions: tailpipe test (HC/CO/NOx) → backpressure (<2 psi) → EVAP smoke

### Key Tests/Specs
- Fuel pressure (35-65 psi key-on/engine-off; 45-55 psi running)
- O2 sensor: 0.1-0.9V switching 8-10x/sec; rich/lean bias
- Injector balance (±5% deviation); coil resistance (0.5-1.5 ohms primary)

### Service Procedures
- EGR: flow test (vacuum pump → gauge rise); clean passages
- EVAP: vent valve click test; purge solenoid resistance (20-40 ohms)
- Catalyst: inlet/outlet temp (>100°F difference); P0420 verify post-drive cycle
- Misfire diagnosis: scope cylinder balance, fuel injector coding
- Fuel trim diagnosis: STFT >+15% (lean) → MAF cleaning/MAF vs calculated
- Catalyst efficiency: pre/post O2 waveforms, temp gun verification
